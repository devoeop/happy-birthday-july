# 🎂 Birthday Surprise Website

A private, personalised birthday website — three pages, full-background images, floating hearts, and a photo gallery.

Built with plain HTML, CSS, and JavaScript. No frameworks, no build step.

---

## Customising

Everything lives in **`config.js`** — open it and edit:
- Her name and all text (headings, letter paragraphs, captions)
- Image filenames for each page's background and the gallery
- Animation settings (speed, heart count, sparkles, opacity)
- Accent colours

See **`images/IMAGES_README.txt`** for where to place your photos.
